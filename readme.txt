===========说 明==============
使用react编写
1、安装node.js
2、安装react
    命令  npm i creat-react-app -g
3、安装react-router-dom
    命令 npm i react-router-dom -S
4、运行 
    命令 npm start
    
========= 实 现 功 能 ===========
1、react模块化         2、SPA页面
3、react-router路由（props.match）
4、webpack打包       5、登录，退出【cookie期限】
6、注册【正则，数组方法】    7、@media响应式 
8、购物车记录保存【localstorage，JSON】
9、购物车编辑【localstorage，JSON】
10、添加购物车登录验证       11、大图预览 
12、详情页图片放大镜        13、取回密码
14、数据筛选         15、无缝轮播图【定时器，h5,css3动画】
16、库存同步【加入删除购物车，库存同步】
17、热卖销量排序（数组方法）
